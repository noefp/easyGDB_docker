# easyGDB_docker
## Installation with Docker

### Step 1:
```
git clone https://github.com/noefp/easyGDB_docker.git
```
### Step 2:
``` 
cd easyGDB_docker/src
```
### Step 3:
```
git clone https://github.com/noefp/easy_gdb.git
```
### Step 4:
```
docker compose up -d
```
At this moment all the features of *easy_gdb* should be already available. In a web browser (Chrome, Firefox, etc.) go to:
```
localhost:8000/easy_gdb/
```
You should be able to see an example of EasyGDB running.
